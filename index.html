FAQ

Questions fréquentes

Trezo fonctionne-t-il sans connexion Internet ?

Trezo Classique (et sa version Windows) fonctionne entièrement hors ligne après installation : idéal si votre connexion est irrégulière. Trezo Cloud nécessite Internet pour synchroniser vos données entre appareils — c'est le prix de la sauvegarde automatique.

Sur quels appareils puis-je utiliser Trezo ?

Téléphone Android ou iPhone, tablette, et ordinateur (Windows, Mac, Linux) : l'application s'installe depuis le navigateur en deux clics. Une version Windows avec installateur classique (.exe) est également proposée.

Mes données sont-elles en sécurité ?

Avec Classique, vos données ne quittent jamais votre appareil — personne d'autre ne peut les voir, pas même nous. Avec Cloud, elles sont hébergées sur une infrastructure sécurisée avec cloisonnement strict par entreprise, et votre compte est protégé par mot de passe. Dans les deux cas, le mode lecture seule à code PIN protège vos chiffres des regards actifs.

Que se passe-t-il quand ma licence expire ?

L'application se met en pause mais vos données sont intégralement conservées. Dès que vous entrez la clé de renouvellement, tout redémarre exactement où vous en étiez.

Puis-je importer mes données existantes ?

Oui. Le module Flux importe vos fichiers Excel (.xlsx) ou CSV : structure contrôlée avant import, erreurs signalées ligne par ligne, aucune ressaisie. Un modèle prêt à remplir est fourni dans l'application.

Trezo gère-t-il ma devise ?

19 devises : FCFA (UEMOA et CEMAC), dirham, dinar, naira, cedi, shilling, franc guinéen, ouguiya, rand, euro, dollar et d'autres. Vous choisissez la vôtre à l'installation.

Puis-je passer de Classique à Cloud ?

Oui, en quelques minutes : exportez la sauvegarde depuis votre installation Classique, importez-la dans votre espace Cloud — flux, stocks, factures, tout est repris.

Y a-t-il une assistance ?

L'application se prend en main sans formation. Pour toute question, l'assistance se fait par WhatsApp — réponse dans la journée ouvrée.

Puis-je donner des accès à mes employés ?

Oui, sur les deux versions. Sur Trezo Cloud, l’administrateur crée directement le compte de chaque collaborateur (e-mail, mot de passe initial, rôle) — ou l’invite par code — avec des droits réglables module par module, et chaque action est tracée dans le journal d’audit. Sur Trezo Classique, des profils locaux avec droits sont disponibles sur le poste, pour l’équipe qui partage l’ordinateur.
